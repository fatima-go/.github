fatima-go는 [golang](https://go.dev/) 언어로 프로그램 개발을 손쉽고 빠르게 진행할 수 있도록 도와주는 개발 프레임워크인 **fatima 패키지**를 제공합니다<BR>
또한 fatima 패키지는 개발한 프로그램을 디플로이하고 모니터링할 수 있는 운영 환경을 제공합니다<BR>
